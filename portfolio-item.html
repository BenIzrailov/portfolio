Spotify SoundSource

Web App powered by Spotify's API

SoundSource is a web application which allows for crowdsourcing playlists throught user suggestions and upvotes.

A user is able to login with their Spotify account and then suggest or vote on songs for three different playlists (General, Party, Working Out).

At the end of each week the songs with the highest upvotes are then added to a central playlist. These playlists can then be easily exported to the users Spotify account with the export and play buttons above the playlist.
